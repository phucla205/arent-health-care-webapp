## Getting Started

### Installation

To use this template

- Clone the project: 
  ```sh
   git clone git@github.com:phucla205/arent-health-care-webapp.git
  ```
- Install dependencies:

  ```bash
  $ yarn install
  # or
  $ npm install
  ```

- Run
  ```bash
  $ yarn start
  # or 
  $ npm run start
  ```

### Project Routes
- Home Page: '/'
- Column Page: '/column'
- My Record: '/record'
